# Raymond Command Center - Memory

This folder is for durable project memory and supporting reference material.

Use it for:
- Approved project context that should survive across sessions.
- Supporting research or reference notes.
- Architecture notes after they are approved.
- Session summaries when they contain durable project memory.
- Source-of-truth references that explain why the project is designed a certain way.

Do not use it for:
- Temporary scratch notes.
- Unapproved old Command Center content.
- Finished deliverables.
- Application source code.
